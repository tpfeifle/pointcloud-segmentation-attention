get_metrics(get_model: Callable, coordinates: tensorflow.python.framework.ops.Tensor, features: Optional[tensorflow.python.framework.ops.Tensor], is_training: tensorflow.python.ops.variables.Variable, bn_decay: tensorflow.python.ops.variables.Variable, labels: tensorflow.python.framework.ops.Tensor, sample_weight: tensorflow.python.framework.ops.Tensor, train: bool, attention_single_layer: int) → Tuple[tensorflow.python.framework.ops.Tensor, tensorflow.python.framework.ops.Tensor, tensorflow.python.framework.ops.Tensor, tensorflow.python.framework.ops.Operation, tensorflow.python.framework.ops.Tensor, tensorflow.python.framework.ops.Operation]¶

gets model metrics

Parameters

  • get_model – method which gives prediction tensor back

  • coordinates – point coordinates (BxNx3)

  • features – feature tensor (optional) (BxNx?)

  • is_training – tf var which indicates if model is training

  • bn_decay – tf var which gives batch norm decay

  • labels – label tensor (BxN)

  • sample_weight – sample weight tensor (BxN)

  • train – bool, which indicates whether these are train metrics

  • attention_single_layer – if this value is not -1 than we use attention instead of the attention_single_layer-th max-pooling layer

Returns

loss, accuracy, predictions(BxNxC), iou_update operation, iou, iou_reset operation
